# Netflix Data Analysis Project

## Overview
This project analyses a Netflix dataset using Python to identify trends in content, genres, ratings, and release years.

## Tools Used
- Python
- Pandas
- Matplotlib
- Excel

## Key Features
- Data cleaning and preprocessing
- Genre frequency analysis
- Rating distribution visualisation
- Movie vs TV Show comparison
- Trend analysis by release year

## Skills Demonstrated
- Data cleaning
- Data visualisation
- Exploratory Data Analysis (EDA)
- Python scripting
